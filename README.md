# 🚗 ParkEase – Smart Parking Management System


## ✨ Key Features

### 🔐 Login System
- Basic authentication system
- Demo Credentials:
  - Username: `admin`
  - Password: `1234`

### 📊 Dashboard
- Displays:
  - Total slots
  - Available slots
  - Occupied slots
  - Occupancy percentage
- Visual parking grid for Zone A and Zone B
- Clickable slots (book or release)

### 🚘 Booking System
- Quick booking option
- Full booking form with:
  - Vehicle number
  - Owner name
  - Phone number
  - Vehicle type
- Automatic time tracking

### 📋 Slot Status Page
- Table view of all parking slots
- Filter options:
  - All
  - Available
  - Occupied

### 🔄 Slot Release
- Click on occupied slot to release
- Confirmation popup (modal)

### 🔔 Notifications
- Toast messages for:
  - Successful booking
  - Errors
  - Slot release updates

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and responsive design  
- **JavaScript (Vanilla JS)** – Functionality and logic
